3 分析化学中的误差与数据处理

3.1 分析化学中的误差

误差有两种表示方法：绝对误差（absolute error, EEE）和相对误差（relative error, ErE_rEr​）。

绝对误差是测量值（measured value, xxx）与真实值（true value, xTx_TxT​）之间的差值，即

E=x−xTE = x - x_T E=x−xT​

相对误差是指绝对误差相当于真实值的百分率，即：

Er=ExT×100%E_r = \frac {E} {x_T} \times 100 \% Er​=xT​E​×100%

绝对误差表示测量值与真实值的接近程度，误差越小，准确度越高；相对误差反映的是误差占真实值的比例，因此在绝对误差相同的情况下，待测组分含量越高，相对误差越小。

对试样进行多次平行测定，此时通常用偏差来衡量所得结果的精密度。偏差（deviation, ddd）表示测量值与平均值（mean, x‾\overline xx）的差值，即

d=x−x‾d = x - \overline x d=x−x

显然这些偏差有正有负，还有一些偏差可能为0。如果将各单次测定的偏差相加，其和应该为0或接近0，即：

∑i=1ndi=0\sum^n_{i=1} d_i = 0 i=1∑n​di​=0

为了表明分析结果的精密度（precision），将各单次测定偏差的绝对值平均，称为单次测定结果的平均偏差（x‾\overline xx）：

d‾=1n(∣d1∣+∣d2∣+⋯+∣dn∣)=1n∑i=1n∣di∣\overline d = \frac {1} {n} (|d_1| + |d_2| + \cdots + |d_n|) = \frac {1} {n} \sum^n_{i=1} |d_i| d=n1​(∣d1​∣+∣d2​∣+⋯+∣dn​∣)=n1​i=1∑n​∣di​∣

平均偏差代表一组测量值中任何一个数据的偏差，没有正负号，在平行测定次数不多时常用平均偏差来表示分析结果精密度。

单次测定结果的相对平均偏差（d‾r\overline d_rdr​）为：

d‾r=d‾x×100%\overline d_r = \frac {\overline d} {x} \times 100\% dr​=xd​×100%

当测定次数较多时，常使用标准偏差（standard deviation, sss）或相对标准偏差（relative standard deviation, RSD, srs_rsr​）来表示一组平行测定值的精密度，单次测定的标准偏差表达式为：

s=∑i=1n(xi−x‾)2n−1s = \sqrt{\frac {\sum\limits^n_{i=1}(x_i - \overline x)^2} {n-1}} s=n−1i=1∑n​(xi​−x)2​​

相对标准偏差也称变异系数：

sr=sx‾×100%s_r = \frac {s} {\overline x} \times 100\% sr​=xs​×100%

偏差也可用全距（range, RRR, 也称极差）表示，它是一组测量数据中最大值与最小值之差。
