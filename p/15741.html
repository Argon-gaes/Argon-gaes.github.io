8.1 The Variational Method Provides an Upper Bound to the Ground-State Energy of a System

We will first illustrate the variational method. Consider the ground state of some arbitrary system. The ground-state wave function ψ0\psi_0ψ0​ and energy E0E_0E0​ satisfy the Schrödinger equation:

H^ψ0=E0ψ0\hat H \psi_0 = E_0\psi_0 H^ψ0​=E0​ψ0​

We can obtain:

E0=⟨ψ0∣H^∣ψ0⟩⟨ψ0∣ψ0⟩E_0=\frac {\langle \psi_0|\hat H|\psi_0 \rangle}{\langle \psi_0|\psi_0 \rangle} E0​=⟨ψ0​∣ψ0​⟩⟨ψ0​∣H^∣ψ0​⟩​

A beautiful theorem says that if we substitute any other function ϕ\phiϕ for ψ0\psi_0ψ0​ into the equation and calculate the corresponding energy:

Eϕ=⟨ϕ∣H^∣ϕ⟩⟨ϕ∣ϕ⟩E_{\phi}=\frac {\langle \phi|\hat H|\phi \rangle}{\langle \phi|\phi \rangle} Eϕ​=⟨ϕ∣ϕ⟩⟨ϕ∣H^∣ϕ⟩​

then EϕE_{\phi}Eϕ​ will be greater than the ground-state energy E0E_0E0​. In an equation, we have the variational principle:

Eϕ≥E0E_{\phi} \ge E_0 Eϕ​≥E0​

where the equality holds only if ϕ=ψ0\phi=\psi_0ϕ=ψ0​, the exact wave function.

The variational principle says that we can calculate an upper bound to E0E_0E0​ by using any trial function we wish. The closer ϕ\phiϕ is to ψ0\psi_0ψ0​ in some sense, the closer EϕE_{\phi}Eϕ​ will be to E0E_0E0​.

We can choose a trial function ϕ\phiϕ such that it depends upon some arbitrary
parameters, α,β,γ,⋯\alpha,\beta,\gamma,\cdotsα,β,γ,⋯, called variational parameters. The energy also will depend upon these variational parameters:

Eϕ(α,β,γ,⋯ )≥E0E_{\phi}(\alpha,\beta,\gamma,\cdots) \ge E_0 Eϕ​(α,β,γ,⋯)≥E0​

Now we can minimize EϕE_{\phi}Eϕ​ with respect to each of the variational parameters and thus approach the exact ground-state energy E0E_0E0​.

（ 变分法依赖于变分原理，即试验函数的能量一定大于等于基态能量。那么对于任意给定的试验函数，总有办法令其尽可能小，尽可能接近ψ0\psi_0ψ0​，则能量就会更接近E0E_0E0​。 ）
