《无机化学与化学分析》章末总结14

第十四章，内容相对较多的主族元素。

14 p区元素（一）

14.1 第13族、第14族和第15族元素概述

p区（p block）包括第13至第18族共6族元素，与s区元素一起被称为主族元素（main group elements）。若使用罗马数字Ⅲ，Ⅳ，Ⅴ，Ⅵ，Ⅶ，Ⅷ来表示p区各族，则本族元素所表现最高正化合价与族数相同。

从元素周期表中来看，金属性变化趋势是从上到下逐渐增强，与电离能从上到下减小的趋势一致。因此处在中间位置元素的单质以及化合物等常作为半导体应用。

三族中15个元素（前六周期）都可达到各自的族氧化态+3、+4和+5，长周期的9个元素还能显示出有一定稳定性的、较族氧化态低2的氧化态。

14.6 氮

氮原子主要形成共价键，但和低电负性金属也能形成离子键如Li3NLi_3NLi3​N、Mg3N2Mg_3N_2Mg3​N2​等。

将大气中的游离氮气转化为氨的过程叫氮的固定（nitrogen fixation）。氮可以在常温下与金属锂反应或在高温下与金属镁反应，产物水解都可获得NH3NH_3NH3​。但工业合成仍沿用哈伯法：

N2(g)+3H2(g)⟶2NH3(g)N_2(g) + 3H_2(g) \longrightarrow 2NH_3(g) N2​(g)+3H2​(g)⟶2NH3​(g)

该反应需在高温、高压和催化剂存在条件下进行，研发常温固氮方法是科学家们所面对的重要挑战。

NH3NH_3NH3​的化学反应可划分为三大类，加合反应、取代反应和氧化反应。

加合反应也就是氨中N原子上孤对电子发生配位的反应，例如Ag(NH3)2+、Cu(NH3)42+Ag(NH_3)_2^+ 、 Cu(NH_3)_4^{2+}Ag(NH3​)2+​、Cu(NH3​)42+​等氨合物。

取代反应可以认为是NH3NH_3NH3​中的H原子被其他基团取代，例如与金属钠反应生成NaNH2NaNH_2NaNH2​等。也可认为是NH3NH_3NH3​作为一个整体取代其他分子的一个部分，例如氨基氯化汞的生成：

HgCl2(aq)+2NH3(aq)⟶Hg(NH2)Cl(s)+NH4Cl(aq)HgCl_2(aq) + 2NH_3(aq) \longrightarrow Hg(NH_2)Cl(s) + NH_4Cl(aq) HgCl2​(aq)+2NH3​(aq)⟶Hg(NH2​)Cl(s)+NH4​Cl(aq)

氨基氯化汞是一个白色沉淀，该反应用于定性检测Hg2+Hg^{2+}Hg2+离子。

氧化反应也就是-3价N的氧化，但是氨的还原性并不强，NH3NH_3NH3​在空气中不能燃烧，只能在纯氧中燃烧生成氮气。工业上用铂催化氨的氧化，生成NONONO用以制备硝酸：

4NH3(g)+5O2(g)⟶Pt4NO(g)+6H2O(g)4NH_3(g) +5O_2(g) \stackrel {Pt}{\longrightarrow} 4NO(g) + 6H_2O(g) 4NH3​(g)+5O2​(g)⟶Pt​4NO(g)+6H2​O(g)

NH3NH_3NH3​中H原子被羟基/氨基取代后分别生成羟胺/肼，它们的主要用途是作还原剂：

N2H4(l)+2H2O2(l)⟶N2(g)+4H2O(l)ΔrHm⊖=−642.2KJ⋅mol−1N_2H_4(l) + 2H_2O_2(l) \longrightarrow N_2(g) + 4H_2O(l) \quad \Delta_rH_m^{\ominus} = -642.2 KJ \cdot mol^{-1} N2​H4​(l)+2H2​O2​(l)⟶N2​(g)+4H2​O(l)Δr​Hm⊖​=−642.2KJ⋅mol−1

2AgBr(s)+2NH2OH(aq)⟶2Ag(s)+N2(g)+2HBr(aq)+2H2O(l)2AgBr(s) + 2NH_2OH(aq) \longrightarrow 2Ag(s) + N_2(g) + 2HBr(aq) + 2H_2O(l) 2AgBr(s)+2NH2​OH(aq)⟶2Ag(s)+N2​(g)+2HBr(aq)+2H2​O(l)

4Fe3+(aq)+2NH3OH+(aq)+5H2O(l)⟶4Fe2+(aq)+N2O(g)+6H3O+(aq)4Fe^{3+}(aq) + 2NH_3OH^+(aq) + 5H_2O(l) \longrightarrow 4Fe^{2+}(aq) + N_2O(g) + 6H_3O^+(aq) 4Fe3+(aq)+2NH3​OH+(aq)+5H2​O(l)⟶4Fe2+(aq)+N2​O(g)+6H3​O+(aq)

它们的氧化产物（N2/N2ON_2 / N_2ON2​/N2​O）离开反应体系，不会向溶液中带入杂质。同时注意到肼还原的反应放出大量热，因此肼以及衍生物可作为火箭发射的推进剂。

叠氮阴离子N3−N_3^-N3−​是个布朗斯特碱，其化合物通常不稳定，易爆炸。例如Pb(N3)2、Hg(N3)2Pb(N_3)_2、Hg(N_3)_2Pb(N3​)2​、Hg(N3​)2​可作为引爆剂；NaN3NaN_3NaN3​被用于安全气囊。

氮有多种氧化物，从+1到+5化合价均可形成，其中最重要的是NO2NO_2NO2​。二氧化氮是一种红棕色气体，溶于水生成硝酸并释放一氧化氮，是工业生产硝酸的重要反应：

3NO2(g)+H2O(g)⟶2HNO3(g)+NO(g)3NO_2(g) + H_2O(g) \longrightarrow 2HNO_3(g) + NO(g) 3NO2​(g)+H2​O(g)⟶2HNO3​(g)+NO(g)

二氧化氮是奇电子化合物，因而显顺磁性，同时也说明了为什么二氧化氮会二聚为无色气体四氧化二氮。

硝酸是重要的工业三酸之一，它的工业化制备经历以下重要步骤：氨的催化氧化；空气氧化一氧化氮；用水吸收二氧化氮制硝酸。

硝酸作为一个强氧化剂，可以氧化许多金属和非金属：

4HNO3(aq)+Hg(l)⟶Hg(NO3)2(aq)+2NO(g)+2H2O(l)4HNO_3(aq) + Hg(l) \longrightarrow Hg(NO_3)_2(aq) + 2NO(g) + 2H_2O(l) 4HNO3​(aq)+Hg(l)⟶Hg(NO3​)2​(aq)+2NO(g)+2H2​O(l)

3I2(s)+10HNO3(aq)⟶6HIO3(aq)+10NO(g)+2H2O(l)3I_2(s) + 10HNO_3(aq) \longrightarrow 6HIO_3(aq) + 10NO(g) + 2H_2O(l) 3I2​(s)+10HNO3​(aq)⟶6HIO3​(aq)+10NO(g)+2H2​O(l)

硝酸的还原产物受多种因素影响，最常见的是NO或NO2NO或NO_2NO或NO2​。由于H+H^+H+的极化作用，使得浓硝酸的氧化性和反应速率都快于稀硝酸，所以反应生成的NO2NO_2NO2​来不及进一步参与氧化变成NONONO就离开体系，因此浓硝酸的还原产物是NO2NO_2NO2​，稀硝酸的还原产物是NONONO，这一点是动力学因素决定的。

浓硝酸的氧化性虽然强，但是依然氧化不了金这样的不活泼金属，但王水（aqua regia）可以氧化金和铂这样的不活泼金属。王水是浓HClHClHCl和浓HNO3HNO_3HNO3​按体积比3：1混合得到的一种混酸，它能溶解金的功效来自于两方面：一是硝酸本身的强氧化性；二是氯离子对金的配位作用增强了金的还原能力。

Au(s)+4H3O+(aq)+NO3−(aq)+4Cl−⟶[AuCl4]−(aq)+NO(g)+6H2O(l)Au(s) + 4H_3O^+(aq) + NO_3^-(aq) + 4Cl^- \longrightarrow [AuCl_4]^-(aq) + NO(g) + 6H_2O(l) Au(s)+4H3​O+(aq)+NO3−​(aq)+4Cl−⟶[AuCl4​]−(aq)+NO(g)+6H2​O(l)

（ 关于王水溶解金的解释，一些比较早版本的书（包括本书），认为是王水中产生的强氧化性物种亚硝酰氯和氯气的氧化作用，但是这种解释在热力学上是说不通的。

参考吉大、武大和南开三校合编的那本蓝皮无机化学（第四版），原文中有：

可以看出，在没有Cl−Cl^-Cl−存在时，硝酸不能氧化金，但是有Cl−Cl^-Cl−存在时，由于形成了[AuCl4]−[AuCl_4]^-[AuCl4​]−增强了Au的还原能力。王水能溶解AuAuAu和PtPtPt不是王水的氧化能力比浓消酸强，而是Cl−Cl^-Cl−的配位使金属的还原能力增强。过去曾认为在王水中生成了氧化能力强的NOClNOClNOCl（氯化亚硝酰）和Cl2Cl_2Cl2​，现在看来，这些观点是不准确的，因为这在热力学上是不可能的。

但是考虑反应不能只看热力学，动力学也要考虑在内。这段话值得商榷，因为新制得王水的颜色是黄橙色的，通常认为就是浓硝酸和浓盐酸反应生成亚硝酰氯的颜色，只是含量比较少，通常不考虑它的影响。目前配位的解释应该是更为广泛接受的。 ）

硝酸盐高温下显示氧化性，加热会分解生成氧气，而采取什么样的分解方式与该金属的亚硝酸盐和氧化物的稳定性有关：

2KNO3(s)⟶Δ2KNO2(s)+O2(g)2KNO_3(s) \stackrel {\Delta}{\longrightarrow} 2KNO_2(s) + O_2(g) 2KNO3​(s)⟶Δ​2KNO2​(s)+O2​(g)

2Pb(NO3)2(s)⟶Δ2PbO(s)+4NO2(g)+O2(g)2Pb(NO_3)_2(s) \stackrel {\Delta}{\longrightarrow} 2PbO(s) + 4NO_2(g) +O_2(g) 2Pb(NO3​)2​(s)⟶Δ​2PbO(s)+4NO2​(g)+O2​(g)

2AgNO3(s)⟶Δ2Ag(s)+2NO2(g)+O2(g)2AgNO_3(s) \stackrel {\Delta}{\longrightarrow} 2Ag(s) + 2NO_2(g) + O_2(g) 2AgNO3​(s)⟶Δ​2Ag(s)+2NO2​(g)+O2​(g)

在分解温度下，K，NaK，NaK，Na等活泼金属的亚硝酸盐仍然稳定；PbPbPb等金属的亚硝酸盐不稳定，氧化物仍稳定；AgAgAg等金属的氧化物也不稳定，则生成单质。

硝酸铵可用作炸药，分解过程是自身的氧化还原：

NH4NO3(s)⟶ΔN2O(g)+2H2O(g)NH_4NO_3(s) \stackrel {\Delta}{\longrightarrow} N_2O(g) + 2H_2O(g) NH4​NO3​(s)⟶Δ​N2​O(g)+2H2​O(g)

与硝酸相比，亚硝酸的酸性和氧化性都要弱，但亚硝酸也可作还原剂使用。亚硝酸只能存在于水中，且易发生歧化：

3HNO2(aq)⟶HNO3(aq)+2NO(g)+H2O(l)3HNO_2(aq) \longrightarrow HNO_3(aq) + 2NO(g) + H_2O(l) 3HNO2​(aq)⟶HNO3​(aq)+2NO(g)+H2​O(l)

亚硝酸用作氧化剂和还原剂两个经典的例子是：

2HNO2(aq)+2I−(aq)+2H3O+(aq)⟶2NO(g)+I2(s)+4H2O(l)2HNO_2(aq) + 2I^-(aq) + 2H_3O^+(aq) \longrightarrow 2NO(g) + I_2(s) + 4H_2O(l) 2HNO2​(aq)+2I−(aq)+2H3​O+(aq)⟶2NO(g)+I2​(s)+4H2​O(l)

5NO2−(aq)+2MnO4−(aq)+6H3O+(aq)⟶5NO3−(aq)+2Mn2+(aq)+9H2O(l)5NO_2^-(aq) + 2MnO_4^-(aq) + 6H_3O^+(aq) \longrightarrow 5NO_3^-(aq) + 2Mn^{2+}(aq) + 9H_2O(l) 5NO2−​(aq)+2MnO4−​(aq)+6H3​O+(aq)⟶5NO3−​(aq)+2Mn2+(aq)+9H2​O(l)

两个反应都可以定量进行，分析化学中用以测量亚硝酸盐的含量。
